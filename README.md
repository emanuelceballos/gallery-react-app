# React Gallery App

This app has a search feature that searches images using the [Unsplash service](https://unsplash.com/documentation).

## Key notes

- Responsive
- CSS handled by Bootstrap 5
- PWA enabled (only in PRD mode)

## Run in dev mode

```
npm i
yarn start
```

## Build for Production

```
yarn build
```
